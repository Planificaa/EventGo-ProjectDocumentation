### 2.3.2. User Task Matrix ###

La User Task Matrix expone las principales actividades que cada User Persona realiza en su rutina diaria. A cada tarea se le asigna un puntaje tanto de frecuencia como de importancia para el usuario. Esta herramienta facilita reconocer cuáles son las tareas realmente cruciales, comprender cómo las ejecutan y detectar los puntos donde el producto puede ofrecer oportunidades de mejora.

En este caso, se toman como base los dos User Persona: organizadores de eventos y anfitriones de eventos. En cada tabla se registran las actividades que realizan para alcanzar sus objetivos, utilizando una escala de tres niveles para evaluar la frecuencia y la importancia: alta, media ybaja.

| Tareas | Organizadores de eventos | | Anfitriones de eventos | |
|-------|---------------------|------------|-------------|------------|
| | **Frecuencia** | **Importancia** | **Frecuencia** | **Importancia** |
| Planificación del evento | Alta | Alta | Alta | Alta |
| Coordinación con proveedores | Alta | Alta | Media | Alta |
| Gestión de invitados | Alta | Alta | Alta | Alta |
| Promoción del evento | Media | Alta | Baja | Baja |
| Supervisión durante el evento | Alta | Alta | Alta | Alta |
| Post-evento | Media | Media | Alta | Media |

### 2.3.4. Empathy Mapping ###

El Empathy Mapping es una herramienta que ayuda a comprender a fondo a los usuarios, analizando no solo sus acciones y necesidades, sino también lo que piensan, sienten, ven, escuchan y expresan. Este esquema permite detectar sus frustraciones, motivaciones y patrones de comportamiento, facilitando el diseño de soluciones más acordes con sus verdaderas necesidades.
